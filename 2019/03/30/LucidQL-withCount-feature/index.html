LucidQL withCount feature

  • 2019-03-30

I have added a new feature to the library withCount

Details about what this feature gives you can be found in the Adonis Lucid docs

https://adonisjs.com/docs/4.0/relationships#_counts

The library docs have been updated as follows

withCount

Accepts an Array of Objects

Lucid’s own documentation for this feature can be found here https://adonisjs.com/docs/4.0/relationships#_counts

The data returned will include a __meta__ property with a count of the records that are related to the baseTable
